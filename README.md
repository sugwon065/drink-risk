# 성향점수매칭(PSM)을 통한 음주와 고혈압 간의 인과관계 추정

성향점수매칭(Propensity Score Matching, PSM)을 활용하여 음주 여부가 고혈압에 미치는 인과적 효과를 분석한 프로젝트입니다.

## 결과물

- [분석 결과 보고서](./psm_project_sugwoncho_clean.html)
- [코드](./code-version.html)

## 주요 내용

- 탐색적 데이터 분석(EDA)
- 성향점수 추정 및 1:1 매칭
- 공변량 균형 점검
- 평균 처치 효과(ATE) 추정
- Bootstrap을 사용한 분산 및 신뢰구간 추정
